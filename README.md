COMP1004 SPA - Meal Tracker

This repo contains all the code related to my SPA submission.

Project Overview:
It is an SPA website that allows the user to record the meals they eat using custom ingredients that the user defines.
It has 2 pages:
- Dashboard: user can create meals and record when they eat them
- Ingredient creation: user defines which nutrients they need and creates their own custom ingredients with nutritional values.

The core flow of the site goes like this:
(1) User creates a nutrient by giving it a name and unit
(2) User creates an ingredient, entering the nutritional information per 100g (or custom amount of grams).
(3) User creates a meal using ingredients
(4) User tells the site they have eaten one of their meals. It will appear in the meal history.
